# Prophet: Pronóstico de Series de Tiempo

Biblioteca de Python para pronóstico automático de series de tiempo, desarrollada por Meta (Facebook).

## Descripción

Análisis de la biblioteca Prophet aplicando conceptos de Probabilidad y Estadística:

- Modelo matemático: y(t) = g(t) + s(t) + h(t) + εₜ
- Pronóstico de demanda diaria con estacionalidad semanal
- Visualización de componentes (tendencia y estacionalidad)
- Intervalos de confianza al 95%

## Ejecución del código Python en Google Colab

El código puede ejecutarse directamente en Google Colab sin instalación adicional.
